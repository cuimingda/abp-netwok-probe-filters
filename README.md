# My custom filter list for Adblock Plus

## Usage

1. 打开`Adblock Plus`的`Options`页面
2. 进入`Advanced`子页面
3. 点击`ADD FILTER LIST VIA URL`按钮
4. 填写`https://raw.githubusercontent.com/cuimingda/adblock-plus-filter-list/refs/heads/main/filters.txt`，并且点击`ADD A FILTER LIST`按钮

## Features

- 默认过滤baidu首页的所有请求，专注于验证网络是否可以正常访问。
